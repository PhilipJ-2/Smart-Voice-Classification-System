# Development Notes

The project was built in stages to make testing and debugging easier.

## Stage Development

1. Microphone and preamp testing
2. Signal filtering and conditioning
3. FFT frequency classification
4. Display and waveform visualization
5. Final integrated system testing

## Key Focus Areas

- Keeping the signal centered around the bias/reference voltage
- Preventing clipping and distortion
- Improving signal stability before FFT processing
- Displaying classification results clearly
- Testing low, mid, and high frequency behavior
