# Project Overview

The Smart Voice Classification System is an embedded audio signal processing project built around the Arduino GIGA R1 WiFi and GIGA Display Shield.

## Project Goal

The goal of this project was to classify microphone input into low, mid, and high frequency ranges using analog signal conditioning and embedded signal processing.

## Main Subsystems

- Microphone input circuit
- LM358 op-amp preamp
- Signal conditioning network
- Arduino analog sampling
- FFT frequency analysis
- Real-time waveform display
- Low/mid/high classification output

## Purpose

This project demonstrates how analog electronics and embedded software can work together to process real-world signals.
