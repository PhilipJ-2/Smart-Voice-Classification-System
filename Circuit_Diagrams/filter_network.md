# Filter Network

This section documents the RC filter and signal routing stage used after the op-amp preamp.

## Purpose

The filter network helps clean and shape the amplified microphone signal before frequency analysis.

## Main Parts

- RC filter components
- Coupling capacitors
- Signal routing resistor
- Arduino analog input
- Classification test nodes

## Notes

The filter stage was used to improve signal stability and make low, mid, and high frequency detection more consistent.
